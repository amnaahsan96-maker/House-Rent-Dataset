🏠 House Rent Dataset Analysis
📌 Project Overview

This project focuses on data preprocessing, cleaning, analysis, and visualization of the House Rent Dataset using Python. The goal is to explore rental trends and understand how factors like size, furnishing status, city, and tenant preference affect house rent.

🧰 Tools & Libraries Used

Python

Pandas – data loading, inspection, cleaning, and manipulation

NumPy – numerical and statistical operations

Matplotlib – data visualization

📂 Dataset

Dataset Name: House Rent Dataset

Source: Kaggle

Format: CSV

🔍 Data Inspection

The following steps were performed:

Viewing first and last rows (head(), tail())

Dataset information (info())

Shape (rows & columns)

Column names and data types

Label-based and index-based selection using loc and iloc

Frequency distribution using value_counts()

Statistical summary using describe()

🧹 Data Cleaning

Checked missing and non-null values

Removed missing values

Filled missing values using interpolation

Identified and removed duplicate records

📊 Numerical Analysis

Mean

Median

Standard Deviation

(Only numerical columns were used for calculations)

📈 Data Visualization

The following visualizations were created:

Line Plot: Rent trend by index

Multiple Line Plot: Rent vs Size

Bar Chart: Furnishing status distribution

Histogram: Rent distribution

Pie Chart: Furnishing status percentage

Scatter Plot: Relationship between Size and Rent

✅ Conclusion

The analysis shows that house rent is influenced by size, furnishing status, and location. Larger and well-furnished houses generally have higher rent. This project demonstrates how Pandas, NumPy, and Matplotlib work together to efficiently analyze and visualize real-world data.
